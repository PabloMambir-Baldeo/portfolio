Refonte d'un site sur un thème aux choix

Pour réaliser un site web, on peut utiliser plusieurs manière : le coder soi-même ou utiliser un CMS, une plateforme pour faciliter la création de site. Mais le plus intéréssant, surtout lorsqu'on utilise une basse de donnée, c'est de le codé soi-même.

Dans un premier projet, j'ai d'abord réalisé un site sur un thème de mon choix. J'ai choisi pour thème la license de jeux vidéo Monster Hunter, car c'est une license que j'apprécie grandement.

Dans un seconde, projet, j'ai réalisé une refonte de ce site : Autant graphique qu'ergonomique. En plus de cela, j'ai rajouté une base de donnée afin de pouvoir permerttre à des visiteurs de mettre des commentaires.

En réalisant ces deux projets, j'ai pu apprendre à organiser un répertoire pour un site Web, à concevoir et implémenter une base de donnée
